How-to launch:

1. Initialize gatsby

    ```shell
    npm init gatsby
    ```
    
        follow "default" scenario

2.  Install dependencies

    ```shell
    npm install
    cd my-gatsby-site/
    npm install
    ```

3.  Run

    from route project directory: 

    ```shell
      npm run develop
    ```

    Your site is now running at http://localhost:8000
